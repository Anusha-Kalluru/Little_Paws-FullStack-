## Project Overview
Little Paws is a dynamic web-based application developed using PHP and MySQL that simulates an online pet product platform. The project demonstrates server-side development concepts such as database connectivity, session handling, and cart-based workflows.

## Key Features
- Dynamic product listing from a MySQL database  
- Add-to-cart functionality using PHP sessions  
- Cart management and order placement  
- Order confirmation page  
- Secure database configuration  
- Simple and clean user interface  

## Technologies Used
- PHP  
- MySQL  
- HTML5  
- CSS3

## Database Setup
1. Create a MySQL database  
2. Import the `little_paws.sql` file  
3. Update database credentials in `config.php`  

## How to Run the Project
1. Install XAMPP / WAMP / LAMP  
2. Place the project folder inside `htdocs`  
3. Start Apache and MySQL  
4. Import the database using phpMyAdmin  
5. Open `http://localhost/Little-Paws/` in your browser  

## Use Case
This project is intended for academic and learning purposes and demonstrates the fundamentals of PHP and MySQL web application development.
